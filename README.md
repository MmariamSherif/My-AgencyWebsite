# MyAgency Website

This is a complete website for **MyAgency**, including multiple pages, animations, and responsive design.  

## Pages
- `index.html` – Landing page
- `about_us.html` – About Us
- `services.html` – Services
- `our_clients.html` – Our Clients
- `blog.html` – Blog
- `careers.html` – Careers
- `contact_us.html` – Contact Us

## Features
- Professional Header & Footer on all pages
- Animations on cards and sections
- Responsive design for mobile, tablet, and desktop
- Linked CSS and JS files for styling and interactivity
- `_redirects` file for Netlify deployment (prevents 404 errors)

## How to view locally
1. Download or clone the repository
2. Open any HTML file in your browser

## Deployment
This project can be deployed on **Netlify** using Drag & Drop or GitHub integration.
